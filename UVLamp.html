    Project Concept

    Gel nails take roughly a minute of concentrated UV light to cure. I plan on making a curing lamp with an adjustable timer.

    Goals

    Create a UV Lamp with a timer so the user knows when the curing is done. Create the circuits and housing myself. Use the smallest micro-controller possible. 3D print the enclosure with phone charger.

    Starting Point

    Prototyping:
    I will create a the schematics and emulate the circuit before development. I will have to emulate an input state for the 7 segment display. The 3D printing will have to wait until I have the circuit developed before I determine where to house them and how to design the enclosure.

    And We’re Off

    Accomplished so far:
    UV LEDs collected and a strip was tested with a chosen power supply. Currently juggling several projects while testing if the ICs I have will work as I want.
